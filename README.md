# BlinkSense`n`nBlinkSense is a Python-based real-time drowsiness detection system that monitors eye movements using computer vision to prevent accidents caused by driver fatigue.`n`n## Features`n`n- **Real-time Monitoring**: Uses webcam input to track eye state in real-time.`n- **EAR (Eye Aspect Ratio) Calculation**: Utilizes precise facial landmarks to determine if eyes are open or closed.`n- **Alert System**: Triggers an audible alert and visual warning when drowsiness is detected (e.g., eyes closed for a specific duration).`n- **Multiple Backend Support**: Compatible with various Windows camera backends (DSHOW, MSMF).`n- **Configurable Thresholds**: EAR threshold and timing can be customized via environment variables.`n`n## Tech Stack`n`n- Python`n- OpenCV`n- dlib`n- imutils`n- NumPy`n- SciPy`n`n## Installation`n`n1. Clone the repository:`n   ```bash`n   git clone https://github.com/Hariprasath-1603/BlinkSense.git`n   cd BlinkSense`n   ````n`n2. Create and activate a virtual environment:`n   ```bash`n   python -m venv .venv`n   .\.venv\Scripts\Activate.ps1`n   ````n`n3. Install dependencies:`n   ```bash`n   pip install -r requirements.txt`n   ````n`n4. Download the facial landmark model:`n   Ensure `models/shape_predictor_68_face_landmarks.dat` is present.`n`n## Usage`n`nRun the main detection script:`n`n```bash`npython Drowsiness_Detection.py`n````n`n### Configuration`n`nYou can customize the detection sensitivity using environment variables:`n- `DROWSY_EAR_THRESH`: Threshold for eye closure (default: 0.28).`n- `DROWSY_SLEEP_SECONDS`: Number of seconds eyes must be closed to trigger alert (default: 3).`n`n## License`n`nThis project is licensed under the MIT License - see the [LICENSE.txt](LICENSE.txt) file for details.
